Firebase Setup Required

To enable real-time synchronization across all devices, please set up Firebase:

  1. Go to console.firebase.google.com and create a new project
  2. Enable Firestore Database (start in test mode)
  3. Go to Project Settings → Your apps → Add Web app
  4. Copy the firebaseConfig object
  5. Replace the placeholder config in this HTML file (search for firebaseConfig)
  6. Redeploy to Vercel

Until configured, the app uses local browser storage (data won't sync across devices).
